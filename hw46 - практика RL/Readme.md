# Итог

**Основные результаты:** Шумы в окружении при выполнении операций агента могут значительно повлиять на качество выполнения задачи. В целом PPO предрасположен для оубчения сглаживанию таких шумов и агент в базовом обучении способен нивелировать их влияние в разумных пределах. При увеличении внешних шумов успешность выполнения задач значительно падает.

Наиболее сильный эффект на работу агента оказывают некачественная калибровка и задержка действия, тем не менее агент с шумом в актере показал способность оперировать лучше остальных в условиях задержек.

Время выполнения кода на GPU в colab суммарно на обучение и оценку составляет около часа.

**Проблемы и ограничения:** Совместимость версий; обучение требует GPU для скорости. Более сложные среды требуют значительно больше времени для обучения, выходящего за рамки требований работы (2 часа на выполнение кода суммарно).

**Возможные улучшения:** Увеличить timesteps, добавить seeds, проверить качество обучения в более сложной среде (PandaPush, PandaStack)
